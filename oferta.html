2. Определения

2.1. Сервис / WAM Agents — технологический сервис Исполнителя, обеспечивающий для каждого Пользователя изолированную вычислительную среду (Контейнер), подключённую к Telegram, с возможностью эксплуатации персонального ИИ-агента на базе больших языковых моделей сторонних поставщиков, в том числе Claude компании Anthropic, PBC (США) и DeepSeek компании Hangzhou DeepSeek Artificial Intelligence Co., Ltd. (Китайская Народная Республика).

2.2. Агент — ИИ-ассистент, настроенный, инструктированный и эксплуатируемый Пользователем внутри Контейнера.

2.3. Контейнер — изолированная Docker-среда, выделяемая Исполнителем каждому Пользователю с усиленными настройками безопасности (отключённые привилегии, read-only корневая файловая система, сброшенные capabilities Linux, выделенные записываемые тома, выполнение от имени non-root).

2.4. Bot Token — токен Telegram Bot API, либо предоставленный Пользователем, либо сгенерированный от его имени с помощью инструментов Исполнителя. Токен принадлежит учётной записи Telegram Пользователя.

2.5. OAuth Token — токен доступа к API Anthropic, выданный для собственной учётной записи Пользователя на claude.ai. OAuth Token используется в Контейнере для вызова API Anthropic от имени Пользователя.

2.6. Подписка — право Пользователя использовать Сервис в течение оплаченного периода.

2.7. Результат работы — любой текст, данные, медиа, решение или действие, сформированное или совершённое Агентом в Контейнере Пользователя.

2.8. Пользователь — физическое лицо, заключившее с Исполнителем настоящий Договор путём акцепта оферты.

2.9. LLM-провайдер — сторонний поставщик больших языковых моделей, к API которого обращается Контейнер Пользователя для выполнения вычислений Агента. На Дату вступления в силу настоящей редакции LLM-провайдерами являются Anthropic, PBC (США) и Hangzhou DeepSeek Artificial Intelligence Co., Ltd. (КНР). Актуальный перечень LLM-провайдеров поддерживается на https://docs.magnets.tg/subprocessors.

2.10. DeepSeek-default tier — тариф Сервиса, в котором по умолчанию вычисления Агента выполняются через API DeepSeek (см. Раздел 10A настоящей Части).

3. Описание Сервиса

3.1. WAM Agents — Telegram-нативный персональный агентский сервис. При активации Исполнитель выделяет Пользователю отдельный Контейнер, подключает его к аккаунту Telegram Пользователя через бота @wamagentsbot и/или бота, созданного от имени Пользователя, и обеспечивает доступ к Агенту внутри Telegram.

3.2. Вычисления Агента выполняются одним из LLM-провайдеров (см. п. 2.9) в зависимости от тарифа и настроек Пользователя. Применительно к каждому потоку трафика к API LLM-провайдера состав, содержание и юрисдикционные условия передачи раскрываются в настоящей оферте отдельно: для Anthropic — в п. 3.2.1, для DeepSeek — в Разделе 10A и Части B Раздел 6.

3.2.1. Поток через Anthropic. При использовании тарифа на основе Anthropic Claude Сервис обращается к API Anthropic, используя собственный OAuth Token Пользователя. Плата за инференс модели взимается Anthropic непосредственно с подписки Пользователя на Anthropic, отдельно от платы за Подписку на Сервис Исполнителя. Применительно к трафику с API Anthropic, осуществляемому через OAuth Token Пользователя, Исполнитель не определяет состав и содержание передаваемых данных; Пользователь признаётся самостоятельным оператором в отношении такой передачи.

3.2.2. Поток через DeepSeek. При использовании тарифа DeepSeek-default (Раздел 10A) Сервис обращается к API DeepSeek, используя общий технический ключ Исполнителя. Применительно к этому потоку Исполнитель является оператором, привлекающим Hangzhou DeepSeek Artificial Intelligence Co., Ltd. в качестве субпроцессора в порядке ч. 3 ст. 6 Федерального закона № 152-ФЗ. Состав передаваемых данных и юрисдикционные условия раскрываются в Разделе 10A настоящей Части и в Части B Раздел 6.

3.3. В Сервис входят: выделение Контейнера и управление его жизненным циклом, базовая конфигурация Агента (политики, навыки, инструменты), мониторинг безопасности, ротация логов в операционных пределах, пользовательская поддержка через команду /support в @wamagentsbot.

OAuth Token аутентифицирует Агента в Anthropic от имени подписки Пользователя на Anthropic. Для тарифа DeepSeek-default OAuth Token Anthropic от Пользователя не требуется; вызов API DeepSeek осуществляется по общему техническому ключу Исполнителя.

10A. Тариф DeepSeek-default

10A.3. Юрисдикция и обработка данных. 10A.4. Функциональные ограничения.
