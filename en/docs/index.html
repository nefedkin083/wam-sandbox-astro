Guide

From bot to agent in four steps.

WAM Agents runs inside Telegram. Four steps: start the bot, talk to your agent on DeepSeek, optionally attach Claude, add the agent to the chats where you work. Everything else happens through conversation and commands inside Telegram.

Step 01 · Install

Open @wamagentsbot.

Hit Start — the bot walks you through onboarding. Accept the data-processing terms, including cross-border transfer of requests to DeepSeek's infrastructure (PRC). Pick «Try 30 days free» (no card) or pay the subscription, and choose a name for your own bot. We create the bot automatically. The agent is ready immediately on DeepSeek — no OAuth, no third-party keys required.

Step 02 · Meet the agent

Talk to the agent.

Open the agent in Telegram and write in plain language — like to a teammate. By default, the agent runs on DeepSeek: text, code, bash, MCP tool calls, multi-turn dialogues. The first few messages are introductions: the agent asks who you are, what you do, how to address you. Then it offers to keep the default persona or tune it to your style — name, tone, boundaries. Any step of the intro can be skipped.

Step 03 · Attach Claude

Optional — add Claude.

If you need vision (images, screenshots), web search, or long-PDF reading — send /claude to your agent. The bot replies with an Anthropic OAuth link — open it, sign in, and send the code back to the chat. From there switch models with /opus, /sonnet, /haiku. Refresh the token later via /login. Claude tokens bill directly on your Anthropic subscription; we store the key only to run your agent. If the auto-auth service is down, the bot offers a manual fallback. This step is optional — DeepSeek default already works.

Step 04 · Workspace upgrade

Set the agent up for multi-tasking.

After the intro, the agent offers an optional upgrade — a dedicated group with forum topics. Each topic becomes an independent session with its own memory: handy for running several projects in parallel without mixing contexts. The agent walks you through setup step by step — create the group, enable chat history for new members, turn on topics, add the bot as admin, grant it permission to manage topics. The step is optional — skip it and come back later anytime.

Bot commands

All work in DMs and groups
/start @wamagentsbot
Start onboarding: subscription or 30-day trial
/status @wamagentsbot
Subscription status and next charge date
/delete @wamagentsbot
Delete the agent and withdraw data-processing consent
/claude in your agent
Attach Claude to your agent via OAuth (optional, on top of DeepSeek)
/login in your agent
Refresh Claude OAuth token (alias for /claude)
/opus in your agent
Opus model — heavy reasoning, long context (after /claude)
/sonnet in your agent
Sonnet model — default balance
/haiku in your agent
Haiku model — fast, cheap answers
/effort in your agent
Reasoning depth: low / medium / high / xhigh / max
/mention in your agent
In groups: reply only on @mention or always
/simple in your agent
Toggle reply mode: plain text or with reasoning
/reset in your agent
New session — clear context
/stop in your agent
Stop current response
/assist in your agent

Media

Voice, photo, video, documents.

The agent reads and sends Telegram files natively. Voice is auto-transcribed. PDFs are read end-to-end — no manual unzip.
