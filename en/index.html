DeepSeek default. Claude optional.

By default, the agent runs on DeepSeek: Q&A, code, bash, MCP tool calls, multi-turn dialogues. Need vision, web search, or long PDFs? Attach Claude with /claude. Then switch models per task: /opus, /sonnet, /haiku.

03

Memory and skills.

The agent remembers your preferences, contacts, projects. Skills and facts can be edited by conversation — just ask it to add, update or forget. Memory lives inside your container.

04

Your data stays yours.

On DeepSeek default, requests flow through a shim into DeepSeek's infrastructure (PRC) — explicit consent is captured during onboarding. Once you attach Claude, chats run through your own OAuth and per Anthropic's BYOK policy, content is not used for model training. We only keep billing metadata and the agent container.
